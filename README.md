# uwmroommatefinder

*Note: Make sure to have Python 3.14 installed as well as all required packages in requirements.txt*  
  
# Useful Commands:  
python manage.py startapp "app name" -> creates an app within the project  
python manage.py migrate -> migrates model changes to the database  
python manage.py runserver -> runs the local server  
python manage.py createsuperuser -> creates an admin account  
  
CTRL-C -> stops local server  
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -> POWERSHELL ONLY - Allows current user to run ps1 scripts for .venv  
  
python manage.py makemigrations -> prepares migrations  
python manage.py migrate -> migrates prepared migrations
